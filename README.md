# go-xcgui-dev

xcgui（炫彩界面库）开发助手技能包。用于 Windows 桌面应用开发。

## 技能目录结构

[下载仓库源码](https://github.com/twgh/go-xcgui-dev/archive/refs/heads/main.zip)后只保留这些文件即可

```
go-xcgui-dev/
├── scripts/
│   ├── search.py        # 源码搜索工具
│   └── download.py      # 源码下载工具
├── SKILL.md             # 技能定义文件
```

## 技能触发场景

- 使用 xcgui 写代码
- 封装新 API
- 查找 xcgui 函数/常量/事件/示例
- 排查 xcgui 编译/运行时问题
- 生成新控件封装代码
